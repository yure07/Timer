# Timer / Cronômetro
<p>Criei um Cronômetro Pessoal simples e intuitivo, o qual pode ser usado para medir o tempo que você realize tais tarefas.</p>
<p> Ele Tem 3 Funcionalidades, Iniciar, Pausar e Zerar.

<h2> Tecnologias Usadas: </h2>
<li> HTML5 </li>
<li> CSS3 </li>
<li> JavaScript </li>
